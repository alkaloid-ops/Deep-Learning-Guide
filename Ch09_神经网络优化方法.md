
# 神经网络优化方法

## SGD

- 经典随机梯度下降

$$
w_t = w_{t-1}-\alpha \frac{\partial J}{\partial w}
$$

```python
torch.optim.SGD(model.parameters(), lr=0.1)
```

## SGD+Momentum

- 在SGD的基础上，引入动量（物理学的惯性）并累计历史梯度

$$
w_t = w_{t-1} - \alpha (\gamma v_{t-1} +\frac{\partial J}{\partial w}) （其中v_{t-1}为累计历史梯度; \gamma为动量, 通常取值0.9）
$$

```python
torch.optim.SGD(model.parameters(), lr=0.1, momentum=0.9)
```

## AdaGrad

- 对历史梯度逐元素平方并求和构成G并开平方根，由学习率除以平方根后的值，再逐元素与当前梯度相乘

$$
w_t = w_{t-1} -\frac{\alpha}{\sqrt{G_{t-1}+\frac{\partial J}{\partial w} \odot \frac{\partial J}{\partial w}} +\epsilon} \odot \frac{\partial J}{\partial w}
$$

$$
（其中G_{t-1}为累计历史梯度；\frac{\partial J}{\partial w}为当前步的梯度；\epsilon为防止分母为0的实数）
$$

```python
torch.optim.Adagrad(model.parameters(), lr=0.01, lr_decay=0, weight_decay=1e-4)
```

## RMSprop

- 使用衰减系数对历史梯度进行遗忘，p与1-p此消彼长的关系来调整历史梯度和当前梯度的构成比例，最后同样逐元素与当前梯度相乘

$$
w_t = w_{t-1}-\frac{\alpha}{\sqrt{pG_{t-1}+(1-p) (\frac{\partial J}{\partial w})^2} +\epsilon}\odot \frac{\partial J}{\partial w}
$$

$$
（其中G_{t-1}为累计历史梯度；\frac{\partial J}{\partial w}为当前步的梯度；\epsilon为防止分母为0的实数；p为衰减系数）
$$

```python
torch.optim.RMSprop(model.parameters(), lr=0.01, alpha=0.99, weight_decay=1e-4)
```

## Adam

- 计算累积梯度的均值和平方累积梯度并指数移动平均，再除以1-衰减系数作为偏差修正，然后学习率除以累积+当前全部梯度并逐元素与梯度均值相乘

$$
\mu_t = \frac{\beta_1 \mu_{t-1}+(1-\beta_1)g_t}{1-\beta^t_1}\quad G_t = \frac{\beta_2 G_{t-1}+(1-\beta_2)g_t^2}{1-\beta^t_2}
$$

$$
（其中\beta_1和\beta_2为衰减系数；\mu_{t-1}为累积梯度均值；G_{t-1}为累计历史梯度；g_t= \frac{\partial J}{\partial w} 为当前步的梯度；\beta^t以时间步数作为指数）
$$

$$
w_t = w_{t-1}-\frac{\alpha}{\sqrt {G_t}+\epsilon }\odot \mu_t
$$

```python
torch.optim.Adam(model.parameters(), lr=0.001, betas=(0.9, 0.999))
```

## AdamW

- 在Adam的基础上增加了权重衰减系数，对上一个时间步的权重进行衰减

$$
\mu_t = \frac{\beta_1 \mu_{t-1}+(1-\beta_1)g_t}{1-\beta^t_1}\quad G_t = \frac{\beta_2 G_{t-1}+(1-\beta_2)g_t^2}{1-\beta^t_2}
$$

$$
（其中\beta_1和\beta_2为衰减系数；\mu_{t-1}为累积梯度均值；G_{t-1}为累计历史梯度；g_t= \frac{\partial J}{\partial w} 为当前步的梯度；\beta^t以时间步数作为指数）
$$

$$
w_t = w_{t-1}-\frac{\alpha}{\sqrt {G_t}+\epsilon }\odot \mu_t-\alpha \lambda w_{t-1}（其中\lambda 为权重衰减系数，即超参数\text {weight decay}）
$$

```python
torch.optim.AdamW(model.parameters(), lr=0.001, betas=(0.9, 0.999), weight_decay=0.01)
```
