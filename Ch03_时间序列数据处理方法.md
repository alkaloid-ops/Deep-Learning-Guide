

## 时间序列数据处理

- 时序数据：记录连续日期下单一观测对象的时序数据（某一支股票、某一个货币的汇率等）
- 面板数据：面板数据记录连续日期下多个观测对象的时序数据
- 截面数据：截面数据是在面板数据中截取某一时间点（极短的时间区间）的时序数据（如某一季度、月份、星期等）包含多个观测对象（是面板数据在时间维度上的切片）

### 单变量与多变量时间序列滑窗

- 单变量是只有日期和目标变量特征两个维度的数据；多变量是除日期和目标变量外还有其他特征维度的数据
- 时间序列数据的日期不可重复，且需要是连续的，中断的日期需要补上，补充的日期的预测标签视为缺失值，并处理缺失值
- 滑窗方式（滑窗前需要将数据集划分为训练集和测试集）：
    - 等宽滑窗：取n个月的数据为训练集，第n+1月的数据为验证集作为第一窗，后续仍保持窗口宽度，往后平移一个月作为第二窗，以此类推
    - 扩展滑窗：取n个月的数据为训练集，第n+1月的数据为验证集作为第一窗，后续将验证集追加至训练集，第n+2月的数据为验证集作为第二窗，以此类推
- 时序数据滑窗代码
    
    ```python
    # 单变量
    def slide(data, window_size):
        x, y = [], []
        for i in range(data.shape[0]-window_size):
            a = data.iloc[i:i+window_size].values
            b = data.iloc[i+window_size]
            x.append(a)
            y.append(b)
    		return torch.FloatTensor(x), torch.FloatTensor(y)
    
    # 多变量(不含历史标签)
    def slide(data, window_size+1):
    		x, y = [], []
    		for i in range(data.shape[0]-window_size):
    				a = data.iloc[i:i+window_size, :-1].values
    				b = data.iloc[i+window_size-1, -1]
    				x.append(a)
    				y.append(b)
    		return torch.FloatTensor(x), torch.FloatTensor(y)
    	
    # 多变量(含历史标签)
    def slide(data, window_size):
    		x, y = [], []
    		for i in range(data.shape[0]-window_size):
    				a = data.iloc[i:i+window_size].values
    				b = data.iloc[i+window_size, -1]
    				x.append(a)
    				y.append(b)
    		return torch.FloatTensor(x), torch.FloatTensor(y)
    ```
    
- 面板数据滑窗代码
    
    ```python
    def paddle_slide(data, window_size):
        x,y = [], []
        for i in np.sort(data['Stock_ID'].unique()):
            single = data[data['Stock_ID'] == i]
            single = single.sort_values(by='Date')
            if single.shape[0] <= window_size:
                continue
            for j in range(single.shape[0]-window_size):
                a = single.iloc[j:j+window_size, :-1].values
                b = single.iloc[j+window_size-1, -1]
                x.append(a)
                y.append(b)
    		return torch.FloatTensor(x), torch.FloatTensor(y)
    ```
    
- 多步预测滑窗代码
    
    ```python
    # 多变量(含历史标签)
    def slide(data, window_size, label_len):
    		x, y, y_index = [], [], []
    		for i in range(data.shape[0]-window_size-label_len+1):
    				a = data.iloc[i:i+window_size].values
    				b = data.iloc[i+window_size:i+window_size+label_len, -1].values
    				c = [*range(i+window_size, i+window_size+label_len)]
    				x.append(a)
    				y.append(b)
    				y_index.append(c)
    		return torch.FloatTensor(x), torch.FloatTensor(y), y_index
    ```
