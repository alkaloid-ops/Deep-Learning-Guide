
# Xavier初始化与He初始化

- Xavier初始化（Glorot条件）：为确保正向传播和反向传播能平稳进行，避免梯度消失和梯度爆炸，要求每一层的输入与输出的方差应尽量保持一致（反向传播的梯度也需要尽量保持一致）。

$$
Var(W)=\frac{2}{n_{in}+n_{out}}（in为输入神经元数；out为输出神经元数）
$$

- He初始化：由于ReLU激活函数会使一半神经元的输出为0，以至于输出方差减半；He初始化修正方差减半，保持输入输出方差一致

$$
正向传播Var(W)=\frac{2}{n_{in}} \quad 反向传播Var(W)=\frac{2}{n_{out}}（in为输入神经元数；out为输出神经元数）
$$

```python
#Xavier初始化(适用于Tanh/Sigmoid激活函数)
torch.nn.init.xavier_uniform_(self.fc1.weight, gain=1.0) #均匀初始化
torch.nn.init.constant_(self.fc1.bias,0) #偏置初始化

torch.nn.init.xavier_normal_(self.fc1.weight, gain=1.0) #正态初始化
torch.nn.init.constant_(self.fc1.bias,0) #偏置初始化

#He初始化(适用于ReLU类激活函数)
torch.nn.init.kaiming_uniform_(self.fc1.weight, a=0, mode='fan_in', nonlinearity='leaky_relu') #均匀初始化
torch.nn.init.constant_(self.fc1.bias,0) #偏置初始化

torch.nn.init.kaiming_normal_(self.fc1.weight, a=0, mode='fan_in', nonlinearity='leaky_relu') #正态初始化
torch.nn.init.constant_(self.fc1.bias,0) #偏置初始化
#其中a为ReLU函数负半轴的斜率；mode参数fan_in为保持正向传播方差；fan_out为保持反向传播方差
```

# 学习率调度（动态学习率超参数）

- 学习率调度可使学习率在训练初期使用较大学习率，到训练后期使用较小学习率来更好收敛模型
- 常用的动态学习率调度有StepLR、ExponentialLR、CosineAnnealingLR、ReduceLROnPlateau等

```python
optimizer = optim.Adam(model.parameters(), lr=0.1)

scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=10, gamma=0.1)            #每隔一定epoch次数(step_size)学习率乘以衰减系数(gamma)
scheduler = optim.lr_scheduler.ExponentialLR(optimizer, gamma=0.95)                  #每个epoch的学习率指数衰减(gamma)
scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=50)                #学习率按余弦曲线从最高下降到最低的步数平滑衰减(初始T_max取值可等于epoch)
scheduler = optim.lr_scheduler.ReduceLROnPlateau(optimizer, factor=0.1, patience=5)  #在多少次(patience)训练后损失不能降的更低就让学习率乘以衰减系数(factor)
```
