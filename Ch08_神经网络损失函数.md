
# 神经网络损失函数

## 回归任务

- 使用均方误差作为损失函数（MSE）

```python
from torch.nn import MSELoss
criterion = MSELoss()
```

## 二分类任务

- 使用二分类交叉熵损失函数（BinaryCrossEntropy）

```python
from torch.nn import BCELoss, BCEWithLogitsLoss
criterion = BCELoss()           #需要在神经网络输出层增加Sigmoid函数将线性预测值转换为概率值
criterion = BCEWithLogitsLoss() #不需要转换为概率值
```

## 多分类任务

- 使用交叉熵损失函数（CrossEntropy）

```python
from torch.nn import CrossEntropyLoss
criterion = CrossEntropyLoss() #不需要转换为概率值
```
